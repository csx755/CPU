# 单周期 RISC-V CPU 整体架构

## 架构概览

本项目实现了一个单周期 RISC-V CPU，所有指令在一个时钟周期内完成。采用哈佛结构（指令存储器和数据存储器分离）。

```
                        +-----------+
                   +--->|    PC     |----+
                   |    +-----------+    |
                   |                     v
                   |              +------------+
                   |              |     IM     | (指令存储器)
                   |              +------------+
                   |                     |
                   |              instr[31:0]
                   |                     |
                   |    +-----------+    |
                   +----|   NPC     |<---+ PC, IMM
                        +-----------+    |
                              ^          |
                              |          v
                        +-----------+  +-----+  +-----+
                        |   ctrl    |<-| Funct7, Funct3, Op | (控制单元)
                        +-----------+  +-----+  +-----+
                              |
              +-------+-------+-------+-------+
              |       |       |       |       |
              v       v       v       v       v
         RegWrite  ALUOp   EXTOp   ALUSrc  WDSel ...
              |       |       |       |
              v       v       v       v
        +-------+  +-------+  +-------+  +-------+
        |  RF   |  | EXT   |  | ALU   |  |  DM   |
        +-------+  +-------+  +-------+  +-------+
              |       |       |               ^
              |       |       |               |
              +-------+-------+---------------+
                      |       |
                      v       v
                  Register Write Data (WD)
```

## 模块层次

```
sccomp (顶层：CPU + IM + DM)
├── PC        (程序计数器)
├── NPC       (下一条指令地址计算)
├── IM        (指令存储器)
├── SCPU      (CPU 核心)
│   ├── ctrl  (控制单元)
│   ├── RF    (寄存器文件)
│   ├── EXT   (立即数扩展)
│   ├── ALU   (算术逻辑单元)
│   └── (WD mux - 写回数据选择)
└── DM        (数据存储器)
```

## 信号命名规范

| 前缀/后缀 | 含义 |
|-----------|------|
| `_w` | 写使能信号 |
| `_sel` | 选择信号 |
| `_op` | 操作码 |
| `rs1/rs2` | 源寄存器地址 |
| `rd` | 目的寄存器地址 |
| `A1/A2/A3` | RF 读/写地址端口 |

## 关键设计约束

1. **单周期**：所有指令在 1 个 clk 周期完成
2. **哈佛结构**：IM 和 DM 分离，互不干扰
3. **同步复位**（部分模块）：复位时寄存器清零
4. **输出显示**：`$display` 监控 DM 写入，便于仿真验证
