# PC — 程序计数器

## 功能描述

PC（Program Counter）是程序计数器，保存当前正在执行的指令地址。每个时钟上升沿，将 NPC（Next PC）的值锁存输出。

## 接口信号

| 信号名 | 方向 | 位宽 | 描述 |
|--------|------|------|------|
| `clk` | input | 1 | 时钟信号，上升沿触发 |
| `rst` | input | 1 | 同步复位，高有效。复位时 PC 清零 |
| `NPC` | input | 32 | 下一条指令地址（来自 NPC 模块） |
| `PC` | output | 32 | 当前指令地址，输出给 IM 和 NPC |

## 内部数据结构

| 名称 | 类型 | 位宽 | 描述 |
|------|------|------|------|
| `PC` | reg | 32 | 存储当前 PC 值，复位为 0 |

## 工作逻辑

```
always @(posedge clk):
    if (rst == 1):
        PC <= 32'b0          // 复位：从地址 0 开始取指
    else:
        PC <= NPC            // 正常：下一个时钟输出新 PC
```

## 连接关系

- **输入来源**：NPC 模块 → NPC 端口
- **输出去向**：IM 模块（取指令）、NPC 模块（计算下一地址）、ALU（JALR 时保存返回地址）
