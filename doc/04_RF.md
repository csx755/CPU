# RF — 寄存器文件

## 功能描述

RF（Register File）是 32 位 × 32 个通用寄存器的寄存器堆。支持双端口异步读（RD1、RD2）和单端口同步写（A3 → WD）。x0 硬连线为 0，不可写入。

## 接口信号

| 信号名 | 方向 | 位宽 | 描述 |
|--------|------|------|------|
| `clk` | input | 1 | 时钟信号，上升沿写寄存器 |
| `rst` | input | 1 | 同步复位，高有效。复位时 x1~x31 清零（x0 始终为 0） |
| `RFWr` | input | 1 | 寄存器写使能，高有效 |
| `A1` | input | 5 | 读端口 1 的寄存器地址（rs1） |
| `A2` | input | 5 | 读端口 2 的寄存器地址（rs2） |
| `A3` | input | 5 | 写端口的目的寄存器地址（rd） |
| `WD` | input | 32 | 写数据，写入 A3 指向的寄存器 |
| `RD1` | output | 32 | 读端口 1 的数据（A1 指向的寄存器值） |
| `RD2` | output | 32 | 读端口 2 的数据（A2 指向的寄存器值） |

## 内部数据结构

| 名称 | 类型 | 位宽×深度 | 描述 |
|------|------|-----------|------|
| `rf` | reg | 32 × 32 | 32 个通用寄存器，`rf[0]` 硬件固定为 0 |

### 寄存器约定（RISC-V ABI）

| 寄存器 | ABI 名称 | 用途 |
|--------|----------|------|
| x0 | zero | 硬连线 0 |
| x1 | ra | 返回地址 |
| x2 | sp | 栈指针 |
| x5-x7 | t0-t2 | 临时寄存器 |
| x8-x9 | s0-s1 | 保存寄存器 |
| x10-x11 | a0-a1 | 函数参数/返回值 |
| x29-x31 | t3-t6 | 临时寄存器 |

## 工作逻辑

**写操作（同步）**：
```
always @(posedge clk):
    if (rst == 1):
        for i = 1 to 31: rf[i] <= 0    // 复位：清零 x1~x31
    else if (RFWr == 1):
        rf[A3] <= WD                    // 写使能：写入数据
```

**读操作（组合逻辑）**：
```
RD1 = (A1 != 0) ? rf[A1] : 0      // x0 始终返回 0
RD2 = (A2 != 0) ? rf[A2] : 0      // x0 始终返回 0
```

## 注意事项

1. **x0 硬连线为 0**：写使能时不应写 x0；读 x0 时永远返回 0
2. **先写后读**：同一周期内写新值时，读端口在仿真中可能读到旧值（组合逻辑读 + 时序逻辑写）
3. **复位只清零 x1~x31**：x0 始终为 0，无需复位

## 连接关系

- **输入来源**：ctrl → RFWr、inst[19:15] → A1 (rs1)、inst[24:20] → A2 (rs2)、inst[11:7] → A3 (rd)、WD mux → WD
- **输出去向**：ALU → RD1, RD2、DM → RD2 (存储数据)
- **调试输出**：reg_data（连接到 sccomp 顶层，可选监控某寄存器值）
