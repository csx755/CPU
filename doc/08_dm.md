# DM — 数据存储器

## 功能描述

DM（Data Memory）是数据存储器，可读可写。在时钟上升沿，若写使能有效，将输入数据写入指定地址。支持 32-bit 字访问（当前版本）。仿真时用 `$display` 打印每次写入操作。

## 接口信号

| 信号名 | 方向 | 位宽 | 描述 |
|--------|------|------|------|
| `clk` | input | 1 | 时钟信号，上升沿写数据 |
| `DMWr` | input | 1 | 数据存储器写使能，高有效（来自 ctrl.MemWrite） |
| `addr` | input | 7 (9:2) | 字对齐的数据地址（字节地址右移 2 位） |
| `din` | input | 32 | 待写入的数据（来自 RF 的 RD2） |
| `dout` | output | 32 | 读出的数据（送至 WD mux） |

## 内部数据结构

| 名称 | 类型 | 位宽×深度 | 描述 |
|------|------|-----------|------|
| `dmem` | reg | 32 × 128 | 数据存储器数组，128 个 32-bit 字，初始值全 0 |

## 工作逻辑

**写操作（同步）**：
```
always @(posedge clk):
    if (DMWr == 1):
        dmem[addr] <= din           // 写入数据
        $display("dmem[0x%8X] = 0x%8X,", addr << 2, din)   // 调试输出
```

**读操作（组合逻辑）**：
```
assign dout = dmem[addr]            // 纯组合逻辑，直接读出
```

## 注意事项

1. **地址是字对齐**：`addr[8:2]` 对应字节地址 `addr[8:2] * 4`
2. **$display 格式**：输出时 `addr << 2` 还原为字节地址，方便对照 ASM 代码
3. **容量**：128 字 = 512 字节，地址范围 0x00 ~ 0x1FC
4. **当前仅支持 word 访问**：DMType 信号已预留但未实现半字/字节访问

## 连接关系

- **输入来源**：ctrl → DMWr、ALU → addr（存储目标地址）、RF → din（RD2，要存储的数据值）
- **输出去向**：SCPU WD mux → dout（用于 LW 写回寄存器）
